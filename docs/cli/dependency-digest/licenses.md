# licenses

View and manage license policies

## Usage

```
dependency-digest licenses
```

## Options

### `--interactive` (-i)

Interactively approve or deny unknown licenses
- **Type:** `boolean`
- **Default:** `false`

- [`allow`](./allow.md) ‚Äî Add licenses to the allowed list
- [`deny`](./deny.md) ‚Äî Add licenses to the denied list
